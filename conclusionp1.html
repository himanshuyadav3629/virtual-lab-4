CONCLUSION


In this practical, we successfully designed and simulated the fundamental logic gates (AND, OR, NOT, NAND, NOR, EX-OR, and EX-NOR) using interactive virtual circuits. Each gate was implemented with:


✅ Interactive Input Switches (On/Off functionality)
✅ Battery Requirement (Preventing direct input activation)
✅ Real-time Output Display (Bulb ON/OFF based on logic operation)
✅ Truth Table Logging (Users can record input-output combinations)/

Through this simulation, we observed how different logic gates function and verified their truth tables practically. This experiment provided a hands-on understanding of how digital logic circuits work, which is essential for designing more complex digital systems.

This virtual lab serves as a great learning tool for digital electronics, allowing users to explore logic gate operations dynamically and interactively.
